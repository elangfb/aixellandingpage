Formula Perhitungan

// ROI Calculation
ROI  = (Total Return − Investasi) / Investasi
     = (3.600M − 1.45M) / 1.45M
     = 148.3%

// Payback Period
PBP  = Grace + (Investasi / CF/bln)
     = 3 + (1.45M / 75Jt)
     = ~22.3 bulan

// Annual IRR (Newton-Raphson)
IRR  = (1 + IRR_monthly)^12 − 1
     = ≈ 58% p.a.

// NPV @ 12% discount rate
NPV  = Σ CF_t / (1+r)^t − Initial
     = + Rp 1.350.000.000 ✅
